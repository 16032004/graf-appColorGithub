<p>1.1 Introducción a la programación de interfaces Web</p>
<p>Este trabajo guía la creación de “una aplicación web de graficación de color” que permite visualizar una tonalidad <strong>RGB</strong> mediante tres controles (rojo, verde y azul), mostrar su <strong>código hexadecimal</strong>, aceptar <strong>entrada RGB en decimal</strong> y sincronizarse con un <strong>color picker</strong>. Se trabaja con <strong>VS Code</strong> y extensiones orientadas a productividad y estilo, y se aplican buenas prácticas de <strong>estructura de proyecto</strong>, <strong>separación de HTML/CSS/JS</strong> y <strong>diseño con Bootstrap</strong>.</p>
<p><strong>Objetivo general:</strong><br>
Configurar un entorno de desarrollo web y construir, probar y embellecer una app que convierta y visualice colores en RGB/Hex, demostrando comprensión de estructura de archivos, maquetado con Bootstrap, manipulación del DOM y validación visual en el navegador.</p>
<p><strong>Herramientas y configuración:</strong></p>
<ul>
<li>
<p>Editor: <strong>Visual Studio Code</strong>.</p>
</li>
<li>
<p>Extensiones: <strong>BlackBox</strong>, <strong>Activitus Bar</strong>, <strong>Live Server</strong>, <strong>Tokyo Night</strong>, <strong>Material Icon Theme</strong>.</p>
</li>
<li>
<p>Estructura de carpetas: <code>Graficacion/1er seguimiento/graf-appColor/</code>.</p>
</li>
<li>
<p>Archivos separados: <code>index.html</code> (plantilla HTML5 con Bootstrap), <code>styles.css</code> (estilos), <code>app.js</code> (lógica).</p>
</li>
</ul>
<p><strong>Desarrollo (síntesis de pasos):</strong></p>
<ol>
<li>
<p><strong>Preparación del documento</strong>: portada institucional con logos (Institución y TecNM), datos de materia, tema, maestro, alumno y fecha.</p>
</li>
<li>
<p><strong>Instalación y tuning</strong> de VS Code con extensiones indicadas.</p>
</li>
<li>
<p><strong>Estructura del proyecto</strong> y alta de la carpeta en VS Code.</p>
</li>
<li>
<p><strong>Plantilla HTML5</strong> y prueba con <strong>Live Server</strong> para verificar visualización.</p>
</li>
<li>
<p><strong>Primera versión de la app</strong>: sliders de R, G y B que actualizan un recuadro de color y muestran el <strong>Hex</strong> correspondiente (HTML/CSS/JS separados).</p>
</li>
<li>
<p><strong>Mejora 1</strong>: campo(s) para <strong>ingresar valores RGB en decimal</strong> que sincronizan el recuadro y los controles.</p>
</li>
<li>
<p><strong>Mejora 2</strong>: <strong>color picker</strong> HTML que refleja su selección en el recuadro y actualiza sliders y valores numéricos.</p>
</li>
<li>
<p><strong>Diseño</strong>: aplicación de <strong>Bootstrap</strong> y estilos propios para un look <strong>moderno y limpio</strong> (alineación, tipografías, tarjetas, espaciados).</p>
</li>
<li>
<p><strong>Verificación continua</strong>: pruebas en navegador tras cada cambio; análisis crítico del código generado (eventos, validaciones 0–255, conversión RGB↔Hex, sincronización de controles).</p>
</li>
</ol>
<p><strong>Producto final esperado:</strong><br>
Una página funcional que:</p>
<ul>
<li>
<p>Muestra un <strong>recuadro de color</strong> en tiempo real.</p>
</li>
<li>
<p>Permite <strong>ajustar RGB</strong> con sliders y/o <strong>ingresar valores numéricos</strong>.</p>
</li>
<li>
<p>Incluye <strong>color picker</strong> sincronizado.</p>
</li>
<li>
<p>Despliega el <strong>código Hex</strong> correspondiente.</p>
</li>
<li>
<p>Presenta un <strong>diseño responsivo</strong> con Bootstrap y estilos propios.</p>
</li>
</ul>
<p><strong>Evidencias sugeridas:</strong></p>
<ul>
<li>Capturas de: estructura de carpetas/archivos, extensiones instaladas, app corriendo en Live Server, funcionamiento de sliders, entrada RGB en decimal, color picker y diseño final.</li>
</ul>
<p><strong>Aprendizajes clave:</strong></p>
<ul>
<li>
<p>Configuración de entorno y extensiones en VS Code.</p>
</li>
<li>
<p>Separación de responsabilidades (HTML/CSS/JS).</p>
</li>
<li>
<p>Manejo de eventos y actualización del DOM.</p>
</li>
<li>
<p>Conversión <strong>RGB ↔ Hex</strong> y validación de rangos.</p>
</li>
<li>
<p>Uso básico de <strong>Bootstrap</strong> para UI moderna y responsiva.</p>
</li>
</ul>

</li>
