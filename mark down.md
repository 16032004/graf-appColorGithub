<p>En esta práctica se desarrolló una <strong>aplicación web interactiva</strong> utilizando <strong>HTML, CSS y JavaScript</strong>, apoyándose en el framework <strong>Bootstrap 5</strong> para la estructura visual y el estilo responsivo.<br>
El objetivo fue <strong>recrear la interfaz de la Bandera de México</strong>, donde el usuario puede manipular las tonalidades de los colores verde y rojo mediante <strong>controles deslizantes (sliders)</strong>, visualizando en tiempo real los valores <strong>HEX</strong> resultantes en cada panel.</p>
<p>Durante el proceso, se creó la estructura del proyecto en <strong>Visual Studio Code</strong>, configurando los tres archivos principales:</p>
<ul>
<li>
<p><strong>index.html:</strong> contiene la estructura y los elementos visuales de la bandera.</p>
</li>
<li>
<p><strong>styles.css:</strong> define los estilos personalizados, tamaños y colores iniciales.</p>
</li>
<li>
<p><strong>app.js:</strong> controla la funcionalidad de los sliders y actualiza dinámicamente los colores.</p>
</li>
</ul>
<p>Posteriormente, se integró el proyecto con <strong>Git y GitHub</strong>, realizando los pasos de inicialización, seguimiento y sincronización de repositorios locales y remotos mediante comandos como <code>git add</code>, <code>git commit</code>, <code>git push</code> y <code>git pull</code>, además de resolver posibles conflictos de merge.</p>
<p>Finalmente, se configuró una <strong>GitHub Page</strong> desde el repositorio remoto para desplegar la aplicación en línea, verificando la correcta visualización desde el navegador y mediante la extensión <strong>Live Server</strong>.</p>
<p>El resultado fue una <strong>interfaz web funcional, limpia y responsiva</strong>, que representa la bandera mexicana con los valores:</p>
<ul>
<li>
<p><strong>Verde:</strong> Esperanza</p>
</li>
<li>
<p><strong>Blanco:</strong> Unidad</p>
</li>
<li>
<p><strong>Rojo:</strong> Sangre de nuestros héroes nacionales</p>
</li>
</ul>
